# 🛍️ Customer Shopping Behavior Analysis Dashboard

## 📊 Overview
This project focuses on analyzing customer shopping behavior to extract meaningful business insights using an interactive Power BI dashboard.

The goal is to understand purchasing patterns, customer segments, and key factors influencing revenue.

---

## 💡 Problem Statement
Businesses often struggle to identify:
- Which category generates the most revenue  
- Whether discounts actually improve sales  
- Customer behavior across gender, season, and location  

This project solves these problems using data-driven insights.

---

## 🔍 Key Insights
- 📈 Electronics category generates the highest revenue  
- 💸 Discounts do not always increase purchase value  
- 👨 Male customers contribute a major share of revenue  
- 🌤️ Seasonal trends show peak sales in Spring & Winter  
- 📍 Certain locations drive higher average revenue  

---

## 📈 Dashboard Features
- KPI Cards (Total Customers, Total Spend, Avg Spend, Ratings)  
- Revenue analysis by category, gender, season, and payment method  
- Top & bottom performing products  
- Shipping type comparison (Average Purchase & Orders)  
- Interactive filters (Category, Gender, Discount, Subscription)
